\CWHeader{Лабораторная работа \textnumero 3}

\CWProblem{
Для реализации словаря (АВЛ-дерево) из предыдущей лабораторной работы, необходимо провести исследование скорости выполнения и потребления оперативной памяти. В случае выявления ошибок или явных недочётов, требуется их исправить.

Результатом лабораторной работы является отчёт, состоящий из:
\begin{itemize}
    \item Дневника выполнения работы, в котором отражено что и когда делалось, какие средства использовались и какие результаты были достигнуты на каждом шаге.
    \item Выводов о найденных недочётах.
    \item Сравнения работы исправленной программы с предыдущей версией.
    \item Общих выводов о выполнении лабораторной работы, полученном опыте.
\end{itemize}

Минимальный набор используемых средств: утилиты \texttt{gprof}, \texttt{gcov} и набор инструментов \texttt{Valgrind} (заменяющий библиотеку \texttt{dmalloc} для расширенного анализа памяти).
}
