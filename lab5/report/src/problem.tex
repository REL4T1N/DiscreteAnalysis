\CWHeader{Лабораторная работа \textnumero 5}

\CWProblem{
Требуется разработать программу, выполняющую линеаризацию циклической строки с использованием суффиксного дерева.

{\bfseries Определение задачи:} Линеаризовать циклическую строку — значит найти минимальный в лексикографическом смысле разрез (циклический сдвиг) данной строки.

{\bfseries Формат ввода}

Входной файл содержит некий разрез циклической строки, состоящий из символов латинского алфавита. Ограничения по длине строки определяются доступной оперативной памятью (вплоть до нескольких сотен тысяч символов).

{\bfseries Формат вывода}

В выходной файл необходимо вывести минимальный в лексикографическом смысле разрез (строку той же длины).

{\bfseries Пример}

\begin{tabular}{p{7cm}p{7cm}}
\textbf{Ввод:} & \textbf{Вывод:} \\
\texttt{xabcd} & \texttt{abcdx} \\
\end{tabular}
}
